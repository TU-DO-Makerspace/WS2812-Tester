## Pins

### RGB Pots

|Arduino Pin|Potentiometer|
|-----------|-------------|
|A1 	    |Red  	  |
|A2 	    |Green	  |
|A3 	    |Blue         |

|Potentiometer|Arduino Pin|
|-------------|-----------|
|Red  	      |A1 	  |

|I2C Pin|Arduino Pin|
|-------|-----------|
|SCL    |A5 	    |
|SDA    |A4 	    |
|VCC    |3.3v 	    |
